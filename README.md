# صلاتي - Islamic App

## ملفات مهمة يجب إضافتها يدوياً

### 1. ملفات الأذكار (assets)
ضع ملفاتك في:
```
app/src/main/assets/morning_adhkar.json
app/src/main/assets/evening_adhkar.json
```

### 2. أيقونة التطبيق
ضع الأيقونات في:
```
app/src/main/res/mipmap-hdpi/ic_launcher.png
app/src/main/res/mipmap-hdpi/ic_launcher_round.png
```

### 3. بناء المشروع
```bash
./gradlew assembleRelease
```

## Package Name
`com.mohamedabdelazeim.islamicapp`
